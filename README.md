# Pocket-Garage
Bu proje, araç sahiplerinin kişisel araçlarını dijital bir garaja kaydetmelerini amaçlamaktadır. Kullanıcılar, bu araçlara ait tüm masrafları (yakıt, servis, bakım, diğer giderler) ve yasal sorumlulukları (muayene tarihi, sigorta tarihi) tek bir platform üzerinden takip edebilecektir.
